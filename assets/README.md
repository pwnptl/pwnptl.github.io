# Assets Structure

Organized asset folders for different media types:

```
public/assets/
├── img/          - Images (jpg, png, webp, svg, etc.)
├── pdf/          - PDF documents
├── doc/          - Documents (docx, xlsx, pptx, etc.)
├── video/        - Video files (mp4, webm, etc.)
├── icon/         - Icon files (favicon, app icons, etc.)
└── font/         - Custom font files (ttf, woff, woff2, etc.)
```

## Usage

### Images
```
/assets/img/gallery-01.png
/assets/img/profile.jpg
/assets/img/logo.svg
```

### PDFs
```
/assets/pdf/resume.pdf
/assets/pdf/document.pdf
```

### Documents
```
/assets/doc/document.docx
/assets/doc/spreadsheet.xlsx
```

### Videos
```
/assets/video/intro.mp4
/assets/video/demo.webm
```

### Icons
```
/assets/icon/favicon.ico
/assets/icon/apple-touch-icon.png
```

### Fonts
```
/assets/font/custom-font.woff2
/assets/font/custom-font.ttf
```

## Best Practices

- Keep file sizes optimized (compress images, minify files)
- Use descriptive file names
- Organize by type as shown above
- Use appropriate formats (webp for images, woff2 for fonts)
